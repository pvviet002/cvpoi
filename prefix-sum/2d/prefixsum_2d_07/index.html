Sự Đa Dạng Của Bức Tranh

Tóm tắt đề bài

Cho lưới N × M với mỗi ô là một ký tự 'a'-'z'. Với mỗi truy vấn (r1, c1, r2, c2), đếm số ký tự khác nhau trong vùng chữ nhật.

Ràng buộc: N, M ≤ 300, Q ≤ 105.

Phân tích

Vì tập ký tự rất nhỏ (chỉ 26 chữ cái), ta có thể đếm sự xuất hiện của từng loại ký tự một cách độc lập.

Với mỗi ký tự k ∈ {'a'...'z'}, ta xây dựng một mảng cộng dồn 2D Sk[N+1][M+1], trong đó Sk[i][j] là số lần ký tự k xuất hiện trong hình chữ nhật (1, 1) đến (i, j).

Khi đó, số lần xuất hiện của ký tự k trong vùng (r1, c1, r2, c2) được tính bằng công thức cộng dồn 2D quen thuộc:

Sumk = Sk[r2][c2] - Sk[r1-1][c2] - Sk[r2][c1-1] + Sk[r1-1][c1-1]

Nếu Sumk > 0, nghĩa là màu k có xuất hiện trong vùng. Ta đếm tổng số ký tự có Sumk > 0.

Minh họa: Tách lớp màu sắc

Chọn một ký tự để xem lưới cộng dồn của nó:

Số lần xuất hiện của ký tự đã chọn trong vùng 2x2 góc trên trái: 0

Tại sao cách tiếp cận này lại hiệu quả với N, M = 300?
Đúng! Với 26 lần tính toán độc lập, tổng số thao tác xấp xỉ 5 triệu, hoàn toàn nằm trong giới hạn thời gian.

Thuật toán từng bước

  1. Tiền xử lý (Xây dựng 26 lưới cộng dồn):
    • Tạo mảng 3D S[26][N+1][M+1].
    • Với mỗi màu k (0 đến 25):
      • Duyệt qua lưới A, nếu A[i][j] == k thì gán val = 1, ngược lại val = 0.
      • Tính S[k][i][j] = S[k][i-1][j] + S[k][i][j-1] - S[k][i-1][j-1] + val.
  2. Xử lý truy vấn:
    • Với mỗi truy vấn (r1, c1, r2, c2):
      • Khởi tạo ans = 0.
      • Lặp k từ 0 đến 25:
        • Tính cnt = S[k][r2][c2] - S[k][r1-1][c2] - S[k][r2][c1-1] + S[k][r1-1][c1-1].
        • Nếu cnt > 0, ans++.
      • In ans.
#include <iostream>
#include <vector>
using namespace std;
int S[26][305][305];
int main() {
int N, M, Q; cin >> N >> M >> Q;
for (int i = 1; i <= N; i++) {
for (int j = 1; j <= M; j++) {
char c; cin >> c;
for (int k = 0; k < 26; k++) {
int val = (c - 'a' == k) ? 1 : 0;
S[k][i][j] = S[k][i-1][j] + S[k][i][j-1] - S[k][i-1][j-1] + val;
}
}
}
while (Q--) {
int r1, c1, r2, c2; cin >> r1 >> c1 >> r2 >> c2;
int ans = 0;
for (int k = 0; k < 26; k++) {
int cnt = S[k][r2][c2] - S[k][r1-1][c2] - S[k][r2][c1-1] + S[k][r1-1][c1-1];
if (cnt > 0) ans++;
}
cout << ans << "\n";
}
}

Độ phức tạp

Thời gian: O(26 × N × M + 26 × Q). Với N, M = 300, Q = 105, số phép tính khoảng ~5 triệu, hoàn toàn trong giới hạn 1 giây.

Cạm bẫy

✕ Quên mảng 1-based

Truy cập S[k][-1] dẫn đến lỗi bộ nhớ.

✓ Padding mảng

Sử dụng mảng [305][305] và bắt đầu từ 1.
