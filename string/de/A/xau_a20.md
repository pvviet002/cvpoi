# Số kế tiếp

## Mô tả đề bài

Cho một xâu biểu diễn một số nguyên không âm. Hãy đổi xâu thành số, cộng thêm $1$, rồi in ra kết quả dưới dạng số.

## Dữ liệu

- Một dòng duy nhất chứa xâu chữ số biểu diễn số nguyên $x$ ($0 \le x \le 10^{18}$).

## Kết quả

- Một dòng duy nhất là giá trị $x + 1$.

## Ví dụ

### Sample input 1
```
1299
```
### Sample output 1
```
1300
```

Số $1299$ cộng thêm $1$ được $1300$.
