# Xâu lặp ký tự

## Mô tả đề bài

Cho một số nguyên dương $n$ và một ký tự $c$. Hãy tạo và in ra xâu gồm đúng $n$ lần ký tự $c$ viết liền nhau.

## Dữ liệu

- Một dòng duy nhất chứa số nguyên $n$ ($1 \le n \le 1000$) và ký tự $c$, cách nhau bởi một dấu cách.

## Kết quả

- Một dòng duy nhất là xâu gồm $n$ ký tự $c$.

## Ví dụ

### Sample input 1
```
5 *
```
### Sample output 1
```
*****
```

Kết quả gồm đúng $5$ dấu `*`.
